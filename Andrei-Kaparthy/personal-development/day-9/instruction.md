# Day 9 
Build and implement GPT-to-terraform pipeline with validation using Checkov and cloudformation Guard.

# project Goal:
Create a pipeline where GPT generates Terraform code, which is then validated by checkov and cloudformation Guard

# Files to create:
-   `main.tf`: The main Terraform file that will be dynamically generated by GPT
-   `gpt_to_terraform.py`: Python script to call GPT generate terraform code , and save it to main.tf
-   `validation.sh`: Shell script to run checkov and cloudformation Guard agianst the generated terraform code
-   `Cloudformation_guard_rules.rules`: Cloudformation rules file.
Checkov_rules.rules: Checkov rules file.
-   `requirements.txt`: lists python dependenies
-   `.gitignore`:Specifies intentionally untracked filed that Git should ignore


# How to run:

-   Set `OPENAI API KEY`:Set your OpenAI API keys as environment variable: `export OPENAI_API_KEY="YOUR_API_KEY"`

-   Install python depencies: `pip install -r requirements.txt`
-   Make `validation.sh` executable: `chmod +x validation.sh`
-   Run the python script: `python gpt_to_terraform.py` (this will generate main.tf)
-   Run the validation script: `./validation.sh`


# checkov and cloudformation guard installation instruction

```sh
pip install checkov
checkov --version
```

Cloudformation Guard: can be installed using `cargo` or by downloading pre-build binaries. Here are the instructions `cargo`:

```sh
cargo install cfn-guard
cfn-guard --version
```
