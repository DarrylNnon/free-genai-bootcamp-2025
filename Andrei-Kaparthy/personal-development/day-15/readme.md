# Day 15 Project: GPT Threat Modeling Bot

Welcome to Day 15 and the start of Week 3! Today's project is to build a "GPT Threat Modeling Bot." This command-line tool will take a system architecture described in a Markdown file, use a sophisticated few-shot prompt to analyze it with an LLM, and generate a comprehensive threat model report based on the STRIDE framework.

## 🚀 The Flow

**Input (Markdown Architecture) → GPT (Few-Shot STRIDE Analysis) → Output (Markdown Report)**
